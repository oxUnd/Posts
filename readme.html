My homepage with org-mode

Introduction

This is the source for my website. It is managed using Emacs and org-mode.

The latest incarnation is based on site.org and weblorg, a wrapper on top of org-export.

It used to be a heavily customized org-publish setup, but weblorg puts all together in a simpler setup and add support for Jinja like templates.

For details about the original implementation see the introductory post and the original source.
